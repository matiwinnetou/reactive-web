Gibralrar project learnings - part 1:

  • latest Google Bot is able to understand at least simple JavaScript, other bots unclear
  • BigPipe - hard to do this right for users, pagelets flickering
  • today's routers optimized for http request and response (http 1.0) - magic citrix (NetScaler) setting - httpcompressionpush=1

Gibralrar project learnings - part 2:

  • monitoring web service calls important - problem may not be in the place where you thought - used Netflix Hystrix
  • possible to use Play with Java but not always idiomatic
  • Play's library - enumerators and iteratees hard to grok, RxJava and Observable may be better for Play Java

Gibralrar project learnings - part 3:

  • Sbt - play's (simple build tool) actually has steep learning curve
  • Play is stable but changes (API breaking changes every two releases)
